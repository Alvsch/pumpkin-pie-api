# Pumpkin Pie API
A wrapper around pumpkin-plugin-api to make our lives better

## Features
- [x] organized and understandable exports
- [x] PumpkinResult instead of pumpkin_plugin_api::Result
- [x] permission constructor 
- [x] event handler wrapper trait
- [x] config trait api
- [x] feature gates for non-plugin api
- [ ] documented exports
